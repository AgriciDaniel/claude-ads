## 2. 機能要件

### 2-1.「自動化」の範囲
以下のどこまでをスコープとするか明確にする。段階が下がるほど難易度・リスクが上がる。

| レベル | 内容 | 例 |
|--------|------|-----|
| Lv.1 レポート自動化 | データ取得→分析→レポート生成 | 日次/週次レポートの自動配信 |
| Lv.2 監査・提案 | アカウントを分析し改善案を提示 | claude-adsの監査チェックに相当 |
| Lv.3 半自動運用 | AIが提案→人が承認→実行 | 入札調整案を提示、承認後に反映 |
| Lv.4 全自動運用 | AIが判断して直接変更を実行 | 入札・予算・配信ON/OFFを自動操作 |

- [ ] どのレベルを求めているか
- [ ] Lv.3以上の場合、承認フローは必要か
